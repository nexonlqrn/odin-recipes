# Odin Recipes

**Odin Recipes** is a simple HTML website built as a beginner-friendly project. It lets you explore recipes for **Famous Chicken Adobo**, **Good Old-Fashioned Pancakes**, and **Tiramisu**.

This project demonstrates the fundamentals of HTML by using essential elements such as headings, paragraphs, lists, links, and images. It also shows how page navigation works through hyperlinks and how to display images within web pages.

By completing this project, you'll gain a basic understanding of creating a multi-page HTML website and organizing content in a clean and structured way.
